위치 검색 앱
==========

## **기능 요구사항**

1. **현재 위치 표시**
2. **주변 관심 장소 마커로 표시**
3. **선택한 장소까지의 경로 표시**
4. **위치 정보 공유 기능**

## **UI 구성**

1. **지도 뷰**
2. **장소 검색 바**
3. **현재 위치로 이동 버튼**
4. **선택한 장소 정보 표시 패널**
